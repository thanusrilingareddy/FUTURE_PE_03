AI SEO Blog & Content Cluster Generator
📌 Project Overview
This project demonstrates how AI + Prompt Engineering can be used to create SEO-optimized blog content for local businesses.

The goal is to generate a complete SEO content system, including a pillar blog and supporting blogs, to improve Google ranking, website traffic, and business visibility.

🏢 Business Selected
Sri Lakshmi Beauty Salon – Eluru
Target Audience: Women (18–40)
Services: Hair care, skincare, bridal makeup

❗ Problem Statement
Many local businesses fail to rank on Google because:

No proper SEO strategy
Random blog content
Lack of keyword targeting
No structured content clusters
💡 Solution
This project builds an AI-powered SEO content system that generates:

Pillar blog (main topic)
Supporting blogs (content cluster)
SEO-friendly structure (H1–H3)
Keyword-focused content
Local SEO optimization
🧠 Prompt Engineering Approach
The system uses structured prompts to generate SEO content.

🔹 Content Structure:
Pillar Blog → Supporting Blogs → Internal Linking

🔹 Framework:
Keyword targeting
Search intent matching
Local SEO (Eluru-based keywords)
Readable and helpful content
